Visualización del Best Server y celdas LTE a partir de un log de Drive Test utilizando Python y QGIS

Publicado el: 24 de Diciembre de 2024

En el campo de las telecomunicaciones, identificar los servidores preferidos (“best servers”) es crucial para entender la distribución y rendimiento de la red. Hoy quiero compartir un script que desarrollé utilizando Python en QGIS para representar gráficamente las muestras de PCI recolectadas en campo y las celdas LTE que comparten el mismo identificador, asignándoles colores coincidentes. Este enfoque automatizado mejora la interpretación de datos y optimiza el proceso de análisis.

El Desafío La visualización de datos de drive test junto con las celdas definidas en una red LTE puede ser un reto cuando se busca identificar las relaciones entre ellos. Este script simplifica el proceso asignando colores iguales a las muestras de PCI y a las celdas LTE correspondientes, lo que facilita la identificación de “best servers”.

A continuación, presento el código dividido en secciones, junto con una explicación de cada bloque:

Importamos los módulos necesarios para interactuar con QGIS y manejar colores en los estilos de capas.

Definimos los nombres de las capas y los campos que contienen los identificadores PCI en cada capa.

Identificamos los índices de los campos relevantes en cada capa.

Obtenemos todas las características (geometrías y atributos) de ambas capas para procesarlas juntas. Extraemos todos los valores únicos de PCI (muestras y celdas) para asignar colores. Generamos colores únicos para cada PCI utilizando un hash del valor. Creamos un renderizador categorizado para la capa de puntos, asignando un color único a cada PCI.

Repetimos el proceso para la capa de polígonos, garantizando que las celdas LTE tengan el mismo color que sus muestras correspondientes. Refrescamos las capas para que los cambios sean visibles en QGIS.

Beneficios del Script

Eficiencia: Automatiza la identificación de “best servers”, reduciendo errores manuales. Claridad Visual: Facilita la interpretación de relaciones entre muestras y celdas. Flexibilidad: Puede ser adaptado para otros casos de uso geoespacial.

Este script demuestra cómo Python y QGIS pueden trabajar juntos para resolver problemas específicos en telecomunicaciones.
